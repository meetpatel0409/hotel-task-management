 Hotel Ops Task System

A hotel operations task management platform that enables Managers, HODs, and Employees to efficiently assign, track, verify, and complete operational tasks with photo-based proof submission.

 Features

- Role-based authentication
- Admin Dashboard
- Manager Dashboard
- HOD Dashboard
- Employee Dashboard
- Task Assignment & Scheduling
- Task Reassignment
- Shift Handover Management
- Photo Proof Uploads
- Task Verification Workflow
- Operational Analytics Dashboard
- Responsive Premium UI

 Technology Stack

- Flask
- SQLAlchemy
- SQLite
- HTML/CSS/JavaScript
- Chart.js
- Flatpickr
- Selenium
- Pytest
